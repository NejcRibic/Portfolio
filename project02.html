The following code has been made by me through my time spent in high school where we were learning about Java. Using the NetBeans Graphical User Interface allowed me to simply take-off and start learning, while also creating some simple applications that to user offer some simple functions. The applications are displayed here for you to either spectate or download. The applications are yet a work in progress or a subject to change thus not everything might not work as expected.
